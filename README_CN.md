# Balatro High Card 模组
一个为 Balatro 引入 X出牌 机制的模组，包含 1 个牌组和 52+ 张 X出牌。

[官方网站](https://www.balatrohighcardmod.com/) 已上线！

[模组预告片](https://www.youtube.com/watch?v=ae5O06sSRlY) 也已发布！

本模组灵感来源于动漫《HIGH CARD》，小丑牌美术与机制均致力于将这两部以扑克牌为主题的热门作品相结合。

# 安装方法

1. 请先按照说明安装 [Steamodded](https://github.com/Steamopollys/Steamodded)（目前支持 0.9.8+，1.0 版本适配进行中）。绝大多数 Balatro 模组都依赖此框架。

快速安装第 2 步：下载 zip 文件并将其解压到 Steamodded 的 Mods 文件夹中（版本号对应 Steamodded 版本），你会看到一个名为 `HighCardMod` 的文件夹；

2. 将 `HighCardMod` 文件夹放入 Steamodded 的 Mods 文件夹即可完成安装。

3. 以**玩家**身份开始你的旅程！

**注意：本模组的首个完整版本发布于 2024 年 5 月 18 日。若你曾游玩过该日期之前的版本，建议重新全新安装！**

# X出牌 与 X出牌牌组

安装模组后，你将看到一个名为 *X出牌牌组* 的新牌组，该牌组在每局开始时拥有 **52 张 X出牌** 和一张传奇小丑牌：

![X出牌小丑](./HighCardMod/assets/2x/j_hcm_xplay.png)![X出牌牌组](./HighCardMod/assets/2x/b_xplaying.png)

**X出牌** 会在你以**高牌**打出并在**本回合第一手**得分后，变身为一张**特殊小丑牌**。

回合结束时，它会**变回** **X出牌**，让你在下一回合决定出什么牌！（少数例外情况除外）

**X出牌小丑**则会在需要时将普通扑克牌转化为 **X出牌**！
